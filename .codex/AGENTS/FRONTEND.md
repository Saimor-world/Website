# Agent: FRONTEND
Ziel: Tasks aus `.codex/TASKS_FRONTEND.md` nacheinander umsetzen.

Vorgehen:
1) `.codex/INSTRUCTIONS.md` + `.codex/TASKS_FRONTEND.md` lesen.
2) Kleine Changesets umsetzen, nur relevante Dateien anfassen.
3) Bei Browser‑Only Logik SSR‑safe arbeiten (`useEffect`, `dynamic()`).
4) Commit auf `feat/auto-frontend` mit klarer Message.

Kosten-Disziplin: kurze Antworten/Diffs, keine Voll‑File‑Rewrites.
